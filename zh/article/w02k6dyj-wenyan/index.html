1. 科目总论

1.1 科目一：经义之试

于算器上答题。

1.2 科目二：场中之试

含诸科目，须低速、短途、精准操御，于宽广安全之专用试场行之。

1.3 科目三：道路之试

驾于公路之上，行超车、转弯、换挡诸事。

1.4 科目四：安行知识之试

于算器上再答题。

2. 术语释义

2.1 挡位

  1. 空挡：齿轮未合，无力传于轮。
  2. 一挡：0~10 km/h
  3. 二挡：10~30 km/h
  4. 三挡：25~45 km/h
  5. 四挡：35~70 km/h
  6. 五挡：60~120 km/h
  7. 倒挡：退行。

2.2 线类

  1. 实线：不可逾越，乃车道之界，限尔行处。
  2. 虚线：可逾越，助尔知当下方位。

2.3 车具与操控

  1. 后镜：琉璃之镜，令尔可察后方左右中。
  2. 手制：令车驻止。
  3. 离合：离发动机与变速箱，动调输出之力。
  4. 制动：以两铁片相摩，令车缓行。
  5. 油门：令发动机生更大之力，加车之速。
  6. 安全带：遇撞时缚尔于当在之处。
  7. 仪表：车之诸般信息，多显于此。
  8. 转向灯：令外界知尔欲往何方。

3. 科目一

3.1 习法

  1. 当参阅：
    1. 本地理论驾校所得书卷
    2. 从淘宝所购书卷
  2. 待尔对所学有大略了解后，便可答题。
  3. 做完全部题之三分之一（总约两千题），便可行模拟之试。
  4. 若能得九十分于百分之中，则可过矣。

3.2 题目纲要

  1. 交通信号
    1. 交警手势
    2. 交通灯
    3. 交通标志
    4. 交通标线
  2. 记分之制
    1. 何以扣分
    2. 常考之内容
  3. 车速
    1. 常道之速
    2. 高速之路速
    3. 能见、车速、跟车之距
  4. 仪表
    1. 机械仪表
    2. 仪表与标志之义
    3. 仪表盘指示标志
    4. 灯光与转向灯
  5. 交通法规
    1. 何以取照及用之何处
    2. 驾驶员管理规
    3. 申照之特殊规定
    4. 驾照记分之制
    5. 超速、超载及新规
  6. 新罚则
    1. 枯燥之事……
  7. 其余常考内容
    1. 危驾之行
    2. 交通事故之类
    3. 何以复分
    4. “两证、两标、一牌”
  8. 智能驾辅系统
  9. 新规定与时限
  10. 车辆登记
    1. 需变更登记之情
    2. 何以申请注销登记
    3. 其余技巧
  11. 何以复分（全篇）
  12. 驾照相关事宜
    1. 勿驾错类之车
    2. 何以升级
  13. 事后救人之法
  14. 行于道上当尽文明
  15. 何以处理事故
  16. 酒后驾车之祸

4. 科目二

今须于实世练习矣。
肌肉之记忆与诸般技巧始发挥作用，唯多练多温方有效。
以下指南为过试之用，取照后当学尔自身实际驾御之法。

4.1 起步

入车时，当行以下步骤：

  1. 系安全带。
  2. 调座椅水平之位：踩离合踏板至最深时，腿弯约三十度内角。
  3. 调座椅垂直之位：以手置头顶与车顶之间，不被卡住为度。
  4. 调后视镜至适位：
    1. 左镜当能窥全轮之缘。
    2. 右镜当置门把于中偏左之位。
  5. 转钥匙至车自检之位，并查：
    1. 仪表灯是否正常
    2. 手刹状态
    3. 顶灯是否熄
  6. 踩离合踏板至最深。
  7. 转钥匙至启动模式持短暂（约二秒，寒时更久），松回，发动机当启。

4.2 前行

欲前行者，如此行之：

  1. 踩离合踏板至最深。
  2. 入一挡。
  3. 抬离合踏板至半联动。
  4. 持离合踏板，感：
    1. 踏板之位
    2. 轻颤之幅
    3. 极限位（发动机熄）何在
  5. 若发动机熄：
    1. 入空挡
    2. 行启动步骤六七
    3. 复从本节始
  6. 松手刹。

4.3 停车

  1. 踩制动踏板启制动。
  2. 速低于当前挡位最低速限时，踩离合踏板至底。
  3. 欲离车者，起手刹。

4.4 行向

  1. 缓缓前行，归于标准之位。
  2. 将车中心对准路之左线。
  3. 同时将中雨刷铆钉（稍偏左处）对准两车辙之中。
  4. 或简想车轮之位，而将轮与路面车辙对齐。
  5. 此技通试皆有用。

4.5 注意

  1. 入车后，即行“起步”。
  2. 若下一科目已有车在其中，当停车待系统认科目已始。
  3. 前科目未毕，亦不当停以乱流程。

5. 科目二诸项

5.1 坡起

此科目需精控离合，易败，须多练。
若发动机于半路熄，踩制动踏板止车而重行相关步骤。

  1. 行“行向”。
  2. 将及坡道时，抬离合踏板增力，至少至离合踏板之三分之二。
  3. 以小幅调离合踏板，同时稳路上之位。
  4. 车头近停车线时，再缓，尽柔。
  5. 左后镜底边触最上线时：
    1. 踩制动踏板至半
    2. 并将离合踏板稍踩深
  6. 起手刹，五秒后松。
  7. 抬离合踏板稍高，缓松制动踏板，并将方向盘稍左转。
  8. 至坡顶时，以制动控速下坡。
  9. 勿忘左转，否则将触下一科目始之右线。

5.2 侧方位停车

  1. 行“行向”。
  2. 方向盘底越停车线后止。
  3. 入倒挡。
  4. 续退至左后镜底边近交叉虚线。
  5. 右满舵。
  6. 再缓；第三纵向虚线右自车身现时，直退。
  7. 轮触纵向虚线时，左满舵。
  8. 车身全入泊位时，止，保轮角。
  9. 亮左转向灯。
  10. 入一挡。
  11. 缓前行。
  12. 右下窗角触右线（左偏四十五度）时，直前。
  13. 约三秒后（车身三分之二出泊位），右转一圈。
  14. 通常需更多右转，自行调之。
  15. 行“行向”。

5.3 直角转弯

  1. 行“行向”。
  2. 标志前亮左转向灯。
  3. 减速。
  4. 方向盘底触交叉虚线，或尔觉前轮正触路中时，左满舵。
  5. 后轮过弯后，行“行向”。

5.4 曲线行驶

  1. 行“行向”。
  2. 以左后镜四分之一偏左位对准左线，保同速行。
  3. 令左后轮保一定距（均七十厘米）。
  4. 左转毕后，直前。
  5. 五秒后，或稍左上移头为参；左线最前端在门把位时，右转一圈。
  6. 保左线最前端在门把位。
  7. 左线尽后，直前。
  8. 行“行向”。

5.5 倒车入库

  1. 行“行向”。
  2. 寻一空泊位。
  3. 近右线，保七十厘米隙。
  4. 肩触泊位之交叉虚线时，右满舵。
  5. 雨刷铆钉触泊矩形左角时，直前。
  6. 后轮越左线时，左满舵。
  7. 以雨刷铆钉对标志杆。
  8. 稍缓，方向盘底边触交叉虚线时止。
  9. 入倒挡。
  10. 退；后镜钮近交叉虚线（五十厘米隙）时，右满舵。
  11. 观右后镜；车身触第三纵向虚线右界时，直退。
  12. 门把中触纵向虚线时，再右满舵。
  13. 观左后镜；车身与左线齐（多一点点）时，直退。
  14. 以经验调位。
  15. 轮中心稍越后线时，止。
  16. 入一挡。
  17. 直前。
  18. 肩触交叉虚线（及线）时，左转一圈半。
  19. 续前，可稍快。
  20. 前轮越交叉虚线时，止。
  21. 入倒挡。
  22. 退，勿转方向盘。
  23. 观左后镜；轮将触左侧第一（镜中）虚线左界延长线之位时，右满舵。
  24. 重行步骤十三。
  25. 重行步骤十四。
  26. 重行步骤十五。
  27. 重行步骤十六。
  28. 重行步骤十七。
  29. 肩触交叉虚线（及线）时，右转一圈。
  30. 后轮触交叉虚线时，右满舵。
  31. 前轮越交叉虚线（另一）后止。
  32. 竣工。

6. 科目三

科目三重实操，路上时多，然仍可述其纲。

6.1 灯光考试

  1. 俟语音播报止后，方行所求之动作。
  2. 下一播报始时，止远光灯之操。
  3. 试程如下：
    1. 熄所有灯光，候试始。
    2. 试始时，首动当开近光灯。
    3. 系统自列中随抽情景，尔当正确应之：
      1. 开危险警示灯、近光灯：靠边止车。
      2. 远光灯切换二次：会车、超车、过危区（急弯、桥、上下坡等）。
      3. 开近光灯：光佳，或近有他车。
      4. 开远光灯：光劣。

6.2 发动机启

  1. 鸣笛二声。
  2. 左转向灯。
  3. 一挡。
  4. 松手刹。

6.3 转弯时

  1. 亮左或右转向灯。
  2. 多看转弯方向之后镜。
  3. 二挡最佳。

6.4 常行之事

  1. 人行道箭头标前踩制动。
  2. 注意公交站处。
  3. 注意学校警示标。
  4. 掉头时止。
  5. 有人挡路时鸣笛。
  6. 转向灯亮至实转间隔四秒。

7. 科目四

与科一相类，然更重道路安行与文明驾御。

7.1 纲要

  1. 识标志、标线、路况。
  2. 驾驶技能：
    1. 用制动、减挡、停车
    2. 复杂路况、恶劣天气驾驶
    3. 高速驾驶、灭火及坠湖（或海）中脱逃之法
    4. 知危险货物、碰撞急救
    5. 恶习当改、礼让校车
