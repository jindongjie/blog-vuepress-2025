# 中国驾照（C1）考试简明指南

## 科目概述

科目一：理论考试，在电脑上做题。

科目二：场地考试，包括多个科目，
要求低速、短距离、精准操控，
在宽阔安全的专用考场进行。

科目三：路考，
在公共道路上驾驶并完成超车、转弯、换挡等动作...

科目四：安全知识考试，在电脑上做更多题目。

### 术语列表：

1. 挡位
   - 空挡（齿轮未连接，无动力传递到车轮）
   - 1挡（0~10km/h）
   - 2挡（10~30km/h）
   - 3挡（25~45km/h）
   - 4挡（35~70km/h）
   - 5挡（60~120km/h）
   - 倒挡（向后行驶）
2. 实线：不能压，车道边界线，限制你能去的范围。
3. 虚线：可以压，帮助你了解当前位置。
4. 后视镜：玻璃镜片让你能观察后方左/右/中。
5. 手刹：让车辆永久停止。
6. 离合器：分离发动机和变速箱，动态调节输出动力。
7. 刹车：通过两块金属片相互摩擦使车辆减速。
8. 油门：让发动机产生更多动力，加速车辆。
9. 安全带：碰撞时将你固定在应在的位置。
10. 仪表盘：车辆信息大多显示在这里。
11. 转向灯：让外界了解你想去哪个方向。

## 科目一

应该参考从当地理论驾校获得的两本指南
以及从淘宝购买的最新出版物。
当你对要学的内容有了大致了解后，就开始做题。
做完全部题目的1/3（总共约2000题），
就可以做模拟考试了，
如果能得到90/100分，就能通过考试。

以下是题目大纲：

1. 交通信号

   3. 交通标志
   4. 交通标线
2. 记分制度

   2. 常见考试内容
3. 速度
   1. 普通道路速度
   2. 高速公路速度
   3. 能见度、速度、跟车距离
4. 仪表

   2. 仪表和标志含义
   3. 仪表盘指示标志
   4. 灯光和转向灯
5. 交通法规
   1. 如何获取驾照及使用范围
   2. 驾驶员管理规定
   3. 申请驾照的特殊规定
   4. 驾照记分制度
   5. 超速、超载及新规
6. 新处罚规定
   1. 无聊的内容...
7. 其他常见考试内容
   1. 危险驾驶行为
   2. 交通事故种类
   3. 如何恢复分数
   4. "两证、两标、一牌"
8. 智能驾驶辅助系统

10. 车辆登记

    2. 如何申请注销登记

## 科目二

现在我们需要在真实物理世界中练习，
肌肉记忆和其他技巧开始发挥作用，
只有多练、多复习才有效。
以下指南是为了通过考试，
拿到驾照后你应该学习自己的实际驾驶方式。

### 起步

当你进入车辆时，应执行以下步骤：
    1. 系好安全带。
    2. 调整座椅水平位置，当你把离合踏板踩到最深时，
       腿部弯曲约30度内角。
    3. 调整座椅垂直位置，当你把手放在头顶和车顶之间时，
       不会被卡住。
    4. 调整后视镜到合适位置。
        4.1 左镜应能看到完整车轮边缘。
        4.2 右镜应把门把手放在中间偏左的位置。
    5. 将钥匙转到车辆自检位置，
       检查所有仪表灯是否正常，
       检查手刹状态，检查顶灯是否关闭。
    6. (!!)将离合踏板踩到最深位置。
    7. 将钥匙转到启动模式保持短暂时间
       （约2秒，寒冷时更长），然后松回，
       发动机应该启动。

### 移动

如果你想向前移动，这样做：
    1. (!!)将离合踏板踩到最深位置。
    2. 换到1挡。
    3. 抬起离合踏板到半联动状态
       （离合器有轻微振动）。
    4. 保持离合踏板，感受踏板位置和振动幅度，
       建立对半联动位置的模糊感觉，
       以及极限位置（发动机熄火）在哪里。
    5. 如果发动机熄火，换到空挡，
       执行启动发动机步骤6-7，
       然后从本节开头重新开始。
    5. 松开手刹。

### 停车

如果你不想从悬崖掉下去，这样做：
    1. 踩刹车踏板启动制动。
    2. 当速度低于当前挡位最低速度限制时，
       将离合踏板踩到底。
    3. 如果你想离开车辆，拉起手刹。

### 行驶方向

缓慢向前移动，回到标准位置，
将车辆中心对准道路左线，
同时将中间雨刷铆钉（稍偏左的位置）
对准两条车辙的中心，
或者简单地想象车轮位置
并将车轮与路面上的车辙对齐，
这个技巧在整个考试中都很有用。

(!!) 注意，当你进入车辆后，
立即执行"起步"！
(!!) 注意，如果下一个科目车辆在其中，
停车等待直到系统认为该科目已开始，
同样如果前一个科目未完成也不应停车。

### 科目一 - 坡道起步

这个科目需要对离合器的精确控制，
容易失败，需要多加练习！
如果发动机在半路熄火，
踩刹车踏板停住车辆然后重做步骤7
    1. 执行"行驶方向"
    2. 当你即将到达坡道时，
       抬起离合踏板增加动力，
       至少到离合踏板的2/3位置。
    3. 以小幅度调整离合踏板，
       要像照顾婴儿一样小心，
       同时稳定在路上的位置，
       执行"行驶方向"
    4. 当车头接近停车线时，
       再慢一些，尽量轻柔，
       真的很容易失败。
    5. 当左后视镜底边碰到最上面的线时，
       踩刹车踏板到一半，
       并将离合踏板稍微踩深一点。
    6. 拉起手刹，5秒后松开。
    7. 将离合踏板稍微抬高一点，
       慢慢松开刹车踏板，
       并将方向盘稍微左转。
    8. 当你到达坡顶时，用刹车控制速度，
       让地心引力把你拉下去。
       别忘了左转，
       否则你会在下一个科目开始时碰到右线。

### 科目二 - 侧方停车

    1. 执行"行驶方向"
    2. 在方向盘底部越过停车线后停车。
    3. 换倒挡
    4. 继续后退直到左后视镜底边
       接近交叉虚线。
    5. 向右打满方向
    6. 再慢一些，当第三条纵向虚线右侧
       从车身出现时，直线后退。
    7. 当车轮碰到纵向虚线时，向左打满方向。
    8. 当车身全部进入停车区域时，
       停车，保持车轮角度。
    9. 打开左转向灯
    10. 换到1挡
    11. 缓慢向前移动
    12. 当右下窗角碰到右线（左偏45度）时，
        直线前进。
    13. 大约3秒后
        （车身2/3出停车区域），
        右转一圈。
    14. 通常需要更多右转，自行调整。
    15. 执行"行驶方向"

### 科目三 - 直角转弯

    1. 执行"行驶方向"
    2. 在标志前打开左转向灯。
    3. 减速。
    4. 当方向盘底部碰到交叉虚线
       （或你认为前轮正在碰到路中间）时，
       向左打满方向。
    5. 后轮通过转弯后，执行"行驶方向"

### 科目四 - 曲线行驶

    1. 执行"行驶方向"
    2. 将左后视镜1/4偏左位置对准左线，
       保持同速行驶。
    3. 让左后轮保持一定距离（平均70cm）
    4. 左转结束后，直线前进。
    5. 5秒后，或稍微左上移头（碰到车门槛），
       当左线最前端在门把手位置时，
       右转一圈。
    6. 保持左线最前端在门把手位置。
    7. 左线结束后，直线前进。
    8. 执行"行驶方向"

### 科目五 - 倒车入库

    1. 执行"行驶方向"
    2. 找一个空的停车位
    3. 靠近右线，保持70cm间距
    4. 当肩膀碰到停车区域的交叉虚线时，
       向右打满方向
    5. 当雨刷铆钉碰到停车矩形的左角时，
       直线前进。
    6. 当后轮越过左线时，向左打满方向。
    7. 将雨刷铆钉对准标志杆
    8. 稍微慢一点，当方向盘底边
       碰到交叉虚线时停车。
    9. 换倒挡。
    10. 后退，当后视镜按钮接近
        （50cm间距）交叉虚线时，向右打满方向。
    11. 看右后视镜，当车身碰到
        第三条纵向虚线右边界时，
        直线后退。
    12. 当门把手中间碰到纵向虚线时，
        再次向右打满方向。
    13. 看左后视镜，当车身与左线对齐
        （多一点点）时，直线后退。
    14. 根据经验调整位置。
    15. 当车轮中心稍微越过后线时，停车。
    16. 换到1挡。
    17. 直线前进
    18. 当肩膀碰到交叉虚线（和线）时，
        左转一圈半
    19. 继续前进，可以快一些。
    20. 当前轮越过交叉虚线时，停车。
    21. 换倒挡。
    22. 后退，不要转方向盘。
    23. 看左后视镜，当车轮即将碰到
        左侧第一条（镜中）虚线左边界
        延长线（向前延伸50cm）的位置时，
        向右打满方向
    24. 重做步骤13
    25. 重做步骤14
    26. 重做步骤15
    27. 重做步骤16
    28. 重做步骤17
    29. 当肩膀碰到交叉虚线（和线）时，
        右转一圈。
    30. 当后轮碰到交叉虚线时，向右打满方向。
    31. 前轮越过交叉虚线（另一条）后停车。
    33. 完成！

## 科目三

科目三更注重实际操作，路上时间更多。
但仍然可以总结一个大纲。

### 灯光考试

(!!!) 等语音播报停止后再执行要求的动作，
下一条播报开始时停止远光灯操作。

    1. 关闭所有灯光等待考试开始。
    2. 考试开始时，第一个动作应是开启近光灯。
    3. 现在系统会从列表中随机抽取情景
       （数字越大优先级越低），
       你应该对这些情景做出正确响应：
        3.1 开启危险警示灯、近光灯：
            靠边停车。
        3.2 远光灯切换两次：
            会车、超车、
            通过危险区域：
            急转弯、桥梁、上下坡等...
        3.3 开启近光灯：
            光线条件好，或附近有其他车辆。
        3.4 开启远光灯：光线条件差。

## 科目四

    与科目一类似，但更注重道路安全
    和文明驾驶。以下是大纲：
    1. 识别标志、标线、路况
    2. 驾驶技能
        2.1 使用刹车、减挡、停车
        2.2 复杂路况、恶劣天气驾驶
        2.3 高速公路驾驶、灭火及
            落入湖（或海）中如何逃生
        2.4 了解危险货物、碰撞急救
        2.5 不良习惯、礼让校车
